\section{How to Use This Roadmap}

\begin{infobox}
\textbf{Learning Approach:} Follow the phases in order. Each phase builds on the previous one. Do not skip Phase 1 and 2 — a strong foundation in Python, SQL, and statistics will make every subsequent topic easier to grasp.
\end{infobox}

\vspace{0.4cm}
\noindent\textbf{Difficulty Legend:}

\begin{tabular}{lll}
  \cellcolor{beginner}\quad\quad & \textbf{Beginner} & Little to no prior knowledge required \\[4pt]
  \cellcolor{intermediate}\quad\quad & \textbf{Intermediate} & Requires Phase 1--2 completion \\[4pt]
  \cellcolor{advanced}\quad\quad & \textbf{Advanced} & Requires solid ML and coding background \\
\end{tabular}

\section{Phase 1 — Foundations \hfill \small\normalfont\color{teal}(Minimum: 4--6 Weeks)}

\subsection{1.1 Python Programming \hfill \small\normalfont\color{darkgray}Beginner | 3--4 Weeks}

\begin{tabular}{>{\bfseries}p{3cm} p{11.5cm}}
\toprule
Goal & Write clean Python code: variables, loops, functions, OOP, file I/O \\
\midrule
Core Topics & Data types, control flow, functions, list comprehensions, modules, classes, error handling, virtual environments \\
Resources & \textit{Automate the Boring Stuff with Python} (free), Python.org docs, freeCodeCamp Python course \\
Practice & Build 3 mini-projects: a calculator, a file organizer, a web scraper \\
\bottomrule
\end{tabular}

\vspace{0.3cm}
\noindent\textit{Key libraries to install:} \texttt{pip, virtualenv, jupyter}

\subsection{1.2 SQL \hfill \small\normalfont\color{darkgray}Beginner | 2--3 Weeks}

\begin{tabular}{>{\bfseries}p{3cm} p{11.5cm}}
\toprule
Goal & Query, filter, aggregate, and join data from relational databases \\
\midrule
Core Topics & SELECT, WHERE, GROUP BY, HAVING, JOINs (inner/outer/left), subqueries, window functions, CTEs, indexes \\
Databases & PostgreSQL (recommended), MySQL, SQLite \\
Resources & SQLZoo (free), Mode SQL Tutorial, LeetCode SQL problems \\
Practice & Solve 30 SQL problems on HackerRank or LeetCode \\
\bottomrule
\end{tabular}

\subsection{1.3 Mathematics \& Statistics \hfill \small\normalfont\color{darkgray}Beginner--Intermediate | 3--4 Weeks}

\begin{tabular}{>{\bfseries}p{3cm} p{11.5cm}}
\toprule
Goal & Understand the mathematical foundations behind ML algorithms \\
\midrule
Statistics & Mean, median, variance, standard deviation, probability distributions, hypothesis testing, p-values, A/B testing, Bayesian basics \\
Linear Algebra & Vectors, matrices, dot products, matrix multiplication, eigenvalues \\
Calculus & Derivatives, chain rule, gradients (for understanding backpropagation) \\
Resources & Khan Academy (free), \textit{Statistics} by Freedman, 3Blue1Brown (YouTube) \\
\bottomrule
\end{tabular}

\section{Final Tips for Success}

\begin{enumerate}[leftmargin=*, label=\textbf{\arabic*.}]
  \item \textbf{Learn by doing.} For every concept, write code immediately. Passive reading is ineffective.
  \item \textbf{Use Kaggle.} It offers datasets, notebooks, competitions, and a community — all free.
  \item \textbf{Build in public.} Push everything to GitHub. Write about your projects on LinkedIn or a blog.
  \item \textbf{Don't skip statistics.} It differentiates good data scientists from people who just run models.
  \item \textbf{Master one cloud platform.} Deep expertise in one (e.g., AWS) is worth more than surface knowledge of all three.
  \item \textbf{Join communities.} Kaggle forums, Reddit r/datascience, Hugging Face Discord, local meetups.
  \item \textbf{Stay consistent.} 1.5--2 hours daily beats 10-hour weekend marathons. Spaced repetition works.
  \item \textbf{Review and revise.} Revisit earlier topics after completing later phases — concepts will click better.
\end{enumerate}
